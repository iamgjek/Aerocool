# vue-electron

> A Vue.js project packaged with electron

## Build Setup

#### install dependencies
```bash
$ npm install
```

#### install electron globally
```bash
$ sudo npm install -g electron
```
##### or
```bash
$ npm install electron
$ ./node_modules/electron/dist/Electron.app/Contents/MacOS/Electron .
```

#### run the application
```bash
$ election .
```
